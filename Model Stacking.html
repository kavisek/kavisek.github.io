Model Stacking

Model Stacking¶

Modelstacking is the process is using a initial model to preprocess additional feature for both your training and testing set. The initial model will take your training data as its input the product an output. The output from the initial model will be appending to training data and processed again threw another model. I have used some housing data below that is involved in regression problem. I will stacking a linear regression model with a K-mean algoorithim. The k-means algorithim will be used as a Featurizer grouping the training data into an additional feature. Afterwards I will pass the training data with the appended clustering information through the main model. Note that most of the code here is borrowed from my linear regression notebook, so there are some similarities.
