Dropping Features

Dropping Features¶

Dropping feature is a common and select tasks that is conduct on most datasets before training your model. I use Pandas to drop the majority of features and observations in my work flow. Below is an example of a column of drop statements in Python. You can find the full documentation on dropping null value on Pandas's documents (link below).

Dropping Data Documentation: https://pandas.pydata.org/pandas-docs/stable/generated/pandas.DataFrame.drop.html

Dropping Columns Inplace¶

By default the inplace parameter for all the drop functions is set to False, but your can pass in opposite Bollean value very easily.

# Copy the DataFrame
df_copy = df.copy()

# Drop the Name column
df_copy.drop('Name', axis=1, inplace=True)
