WordClouds in Python

WordClouds in Python¶

A word cloud can be useful when looking at a segmented section of your data. There is a package called word cloud in Python that can create word cloud given a string input. The package gives you the ability to tune the image height, width, and pixel density of the image before exporting the image to your computer.

Create Wordcloud¶

In [10]:
wordcloud = WordCloud(background_color="black", max_words=200,
                     width=700, height=500, margin = 20).generate(text)
plt.imshow(wordcloud, interpolation="nearest");
