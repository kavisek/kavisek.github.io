WordClouds in Python

WordClouds in Python¶

I hate word clouds but sometimes bosses just want something nice to look at. There is a package called word cloud that can create word cloud given a string input. You tune the image height, width, and pixel density of the image before exporting.

Preliminaries¶

In [6]:
%matplotlib inline

import pandas as pd
import matplotlib.pyplot as plt
from wordcloud import WordCloud, get_single_color_func

# Config dataframe and plots
pd.set_option('max_columns', 999)
plt.rcParams["figure.figsize"] = [30,30]
%config InlineBackend.figure_format = 'retina'

Create Text String¶

In [1]:
text = '''In computer programming, pandas is a software library written for the Python programming language for \
data manipulation and analysis. In particular, it offers data structures and operations for manipulating numeri\
cal tables and time series. It is free software released under the three-clause BSD license.[2] The name is der\
ived from the term "panel data", an econometrics term for multidimensional, structured data sets.'''

Create Wordcloud¶

In [10]:
wordcloud = WordCloud(background_color="black", max_words=200,
                     width=2400, height=1600, margin = 20).generate(text)
plt.imshow(wordcloud, interpolation="nearest")
