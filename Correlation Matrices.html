Correlation Matrices

Correlation Matrices¶

Pandas has this great dataframe correlation function that can operate on the entire dataframe. The function support Pearson, Kendall, and Spearman correlations. You can use this function with seaborn to create a heat map to view the data at a glance. For prettier correlation plots you can always just use seaborn but you will not be able to extract the model variables from Seaborn. In that case, there is always Scikit Learn and Scipy for the gritty numbers if you need them.

Import Preliminaries¶

In [15]:
# Import Modules
import numpy as np
import pandas as pd
import seaborn as sns

# Create DataFrame
df = pd.DataFrame(data = {'Feature 1': np.random.randint(0,100, size=10),
                 'Feature 2': np.random.randint(0,100, size= 10),
                 'Feature 3': np.random.randint(0,100, size=10),
                 'Feature 4': np.random.randint(0,100, size=10)})
