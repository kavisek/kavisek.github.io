Linear Regression¶

Below is an example of model using linear regression to predict house prices from a datasets provided on Kaggle. The dataest can be found at the link below. Note that I have tried to reducet the amount of regularization found in sklearn's implementation of the model. As well as atttempting to compare and contrast the performance of both statsmodel's linear regression model and sklearn's linear regression model. I try to avoid a lot of feature engineering here, as this notebook is just an example of the models.

It seems that we have wide range of null valuse for each feature in our datasets. For this example, I will quickly go through each feature and a replace any missing categorical value with the value 'Unknown" and replace any missing categorical valuse with the mean. This will let us quickly clearn the data so we can go on and move onto our linear regression model.
