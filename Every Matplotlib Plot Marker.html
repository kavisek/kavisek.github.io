Every Matplotlib Plot Marker

Every Matplotlib Plot Marker¶

Matplotlib has many markers for its line plots. Sometimes is hard to visualize each marker from just its line marker. Therefore I created a quick notebook to plot each marker out. I often come back to these plot to refresh my memory.

Marker documentation: https://matplotlib.org/api/markers_api.html

Implement Preliminaries¶

In [40]:
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

markers = ['.', ',', 'o', 'v', '^', '<', '>', '1',
           '2', '3', '4', '8', 's', 'p', 'P', '*', 'h',
           'H', '+', 'x', 'X', 'D', 'd', '|', '_', '$yolo $',
           'None']

Visualize Matplotlib Markers¶

In [43]:
for mark in markers:
  data = pd.DataFrame(np.random.choice([1,2,3,4], replace=True, size=10), 
                      columns=['Col A'])
  plt.plot(data.index,data['Col A'], marker=mark, 
           color='purple', markersize=12)
  plt.title(str(mark)+" "+'Marker Plot')
  plt.ylabel('Y Label')
  plt.xlabel('X Label')
  plt.show()
